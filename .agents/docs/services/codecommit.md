## Official AWS Documentation Research

Sources:
- https://docs.aws.amazon.com/codecommit/latest/userguide/welcome.html
- https://docs.aws.amazon.com/codecommit/latest/userguide/getting-started.html
- https://docs.aws.amazon.com/codecommit/latest/userguide/auth-and-access-control.html

Research outcomes:
- CodeCommit is a managed private Git repository service. Repositories contain Git commits, branches, tags, and files.
- Git operations use standard Git clients over HTTPS or SSH while repository management operations use AWS APIs.
- IAM policies control repository access, including Git pull, Git push, branch, tag, and management permissions.
- Users can create repositories, clone them locally, make commits, push branches and tags, and share repository access through IAM permissions.
- CodeCommit supports pull requests, approval rules, comments, notifications, and repository triggers as collaboration features.
- Repository events can integrate with Developer Tools notification rules and other AWS services.

Parity implications:
- Model repositories, commits, trees, blobs, branches, tags, pull requests, approval rules, comments, triggers, and IAM-controlled Git permissions separately.
- Git data-plane state should be distinct from repository control-plane metadata.
- Branch and tag operations should reflect Git object reachability rather than simple key/value storage.

## HTTP Bindings

Per-operation input members that bind to HTTP transport surfaces. Optional members are easy to miss because they do not appear in the operation matrix's Required input column. RFC 7232 conditional headers (`If-Match`, `If-None-Match`, `If-Modified-Since`, `If-Unmodified-Since`) and service-specific modifier headers (`x-amz-*`, `x-amzn-*`) surface here. Every handler must list each binding as honoured, intentionally unsupported, or ignored-with-rationale.

_No `@httpHeader`, `@httpQuery`, `@httpPrefixHeaders`, or `@httpPayload` input members are modelled for this service (typical for `awsJson1_*` protocols, where all input flows through the JSON body)._
