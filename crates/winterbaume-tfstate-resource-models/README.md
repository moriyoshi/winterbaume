# winterbaume-tfstate-resource-models

Auto-generated Terraform resource projection models for winterbaume converters

This crate is part of the [winterbaume](https://github.com/moriyoshi/winterbaume) workspace — a suite of in-process AWS service mocks for Rust. Use the umbrella [`winterbaume`](https://crates.io/crates/winterbaume) crate to pull in all services at once, or depend on this crate directly.

The per-service modules in this crate are generated by `tf-converter-codegen gen <service>` from the specs under `crates/winterbaume-terraform/specs/` and must not be edited by hand.
